Your First Hackathon Project: How to Build a Personal Website

So it’s your first hackathon and you really want to be productive and build something right? You can not go wrong with building a personal website which you can then flesh out and use as the base of your hack.

This is a great place to start because:

  • You will be able to directly interact with HTML/CSS
  • You will learn how to work with the open source development platform, Github
  • Your website will automatically be hosted at no cost

I believe there is no better teacher than experience so we will just jump right in!

Build the Site

We are going to build and host the website with the help of Github.

  1. First go to https://github.com/ and register for an account
  2. Now follow this quick and simple guide from Github begin creating your site. Be sure to follow the steps for the User or organization site.
  3. Great! Now you have a bare bones static site. Time to make it useful. Let’s go back to the github pages site, but this time click and follow the instructions on the tab for Project Site.
  4. Now you have a nice looking website with simple HTML and CSS code for you to customize! You may have noticed having to type a bunch of commands like add, commit and push. You are using the version control system git to manage the files at your created repository. Check out this quick tutorial which explains in depth what each command does and more!

Customization

What’s next? I would mess around more with the HTML and CSS so you can get a feel for how they work. You could make lists, change fonts, embed a widget and so much more!

I recommend looking at the guides from w3schools for some inspiration. Remember that what you can do with HTML and CSS is only define the layout and appearance of your website. If you want to program the behaviour of the website to add some interactivity, look into the JavaScript guide. If you want to input or retrieve data you should look into the SQL section.

I would personally start with the Javascript guide since you can start working with it from everything we just set up.

Free domain name for college students

Notice the url of the site and how it has "github.io" at the end. As a college student we can change that for free! Head over to the Namecheap for education site and claim a free domain to customize your site with.

You are now able to use the domain to represent your website’s address. Follow these instructions from Github to connect your github site with the domain name.

Fin

Congrats! You now have your own hosted static website. It is yours to hack with and develop some web dev skills. This is a great start to completing a web dev hack or if you just want to have a website to show off your project.
