Amazon Alexa Workshop Installation Instructions

Install Python

This workshop will require Python. If you do not have Python set up or do not know how to access it follow this quick tutorial: https://learnpythonthehardway.org/book/ex0.html

Install pip

pip is a package management system used to install and manage software packages written in Python. Depending on your version of Python, pip may or may not already be installed. Installation instructions can be found here: https://pip.pypa.io/en/stable/installing/

Install and activate virtualenv

virtualenv is a tool to create isolated Python environments. Since we will be using Python packages in this workshop, it is strongly recommended that you install the packages in virtualenv to avoid messing up with your global Python packages. You can install virtualenv by issuing the following commands in your terminal: 

pip install virtualenv

Create a new folder for your application: 

mkdir alexa-workshop 
cd alexa-workshop

Create virtualenv in your folder: 

 virtualenv venv

Activate virtualenv: 

 source venv/bin/activate
You will know that you are working in virtualenv as the prompt of your shell/terminal will change to show the active environment. It should look similar to this: 
 (venv)$

Install packages

We will be using the framework Flask-Ask to build our Alexa Skill. Enter the following into your command line to install necessary packages 

pip install flask
pip install flask-ask

Exit virtualenv

After installing packages, you can exit the virtualenv by typing: 

deactivate

Download ngrok

ngrok is a software that establishes secure tunnels from a public endpoint to a locally running network service, and will be used for testing the Alexa skill. Download ngrok for your machine and be sure to unzip the file to a location you can remember: https://ngrok.com/download

Sign up for Amazon Developer

Finally, create an acconunt with Amazon Developer: https://developer.amazon.com/
