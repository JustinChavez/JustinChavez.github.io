Amazon Alexa Workshop Installation Instructions

I will go over all you need to prepare before the Amazon Alexa Workshop at HackPrinceton. If you have any questions please place them in the comments section below.

This workshop will require Python. If you do not have Python set up or do not know how to access it follow this quick tutorial!: https://learnpythonthehardway.org/book/ex0.html

We will be using the framework Flask-Ask to build our Alexa Skill. Enter the following into your command line to install Flask-Ask onto your system.

pip install flask-ask

Note: Depending on your version of Python, pip may or may not already be installed. Installation instructions can be found here: https://pip.pypa.io/en/stable/installing/

Finally download ngrok for your machine and be sure to unzip the file to a location you can remember: https://ngrok.com/download

All done! I hope to see you at the workshop. If you would like to keep updated about all beginner friendly, developer oriented information related to building Alexa Skills sign up for our organizations mailing list below! We promise not to spam!
